# Clean Raid Arena Frames

Minimal WoW addon for cleaner raid and arena unit frames.

## Features

- **Gradient Overlay** - Adds a subtle depth gradient on raid and arena healthbars
- **Clean Arena Accessories** - Repositions and resizes CC remover, debuff, and diminish tray on arena frames
- **Hide Casting Bar** - Hides the casting bar on arena frames for a cleaner look
