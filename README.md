# Super Raid Frames

WoW addon for cleaner raid and arena unit frames.

## Features

- **Gradient Healthbars** — Subtle depth gradient overlay on raid and arena healthbars
- **Healer Buff Glow** — Golden swirl highlight on tracked healer buffs (Atonement, Beacon of Light, Beacon of Faith, Beacon of Savior, Soothing Mist, Lifebloom, Reversion, Earth Shield)
- **Defensive Buff Glow** — Green swirl highlight on big defensive cooldowns, repositioned to top-left
- **Debuff Limiting** — Shows only one debuff per raid frame, enlarged when dispellable CC
- **Buff Grid** — Repositions buffs in a bottom-right grid to compensate for upscaled icons
- **Clean Arena Accessories** — Repositions CC remover, debuff frame, and diminish tray on arena frames
- **Hidden Casting Bar** — Hides the casting bar on arena frames for a cleaner look
