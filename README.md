# Clean Raid Arena

Minimal WoW addon for cleaner raid and arena unit frames.

## Features

- **Gradient Overlay** - Adds a subtle depth gradient on raid and arena healthbars
- **Debuff Limiting** - Shows only one debuff per raid frame to reduce clutter
- **Healer Buff Glow** - Golden swirl highlight on tracked healer buffs (Lifebloom, Soothing Mist, Beacon of Light, Beacon of Faith, Beacon of the Savior)
- **Clean Arena Accessories** - Repositions CC remover, debuff, and diminish tray on arena frames
- **Hide Casting Bar** - Hides the casting bar on arena frames for a cleaner look
